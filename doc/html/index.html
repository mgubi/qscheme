QScheme Documentation

Daniel Crettol

June 12 2000

Abstract:

This is my small Scheme interpreter. The goal of this work is to provide a really fast and portable implementation of the Scheme language. My Scheme should also be easy to use by itself or linked with any program needing an embedded language.
